Ever After Café Folder Maker

## Previous Version
My previous Folder Maker App could:
- Create the main project folders for the Ever After Café project.
- Create a README.txt file with project information.
- Display a success message after creating the project.

## New Version

- NuGet/package use: Installed and used the Newtonsoft.Json package to serialize data into JSON format.
- File save location: Saves CafeData.json inside the café project folder created by the app.

  - Purpose: Creates the folder structure for the Ever After Café project.
  - Inputs: Café name

  - Purpose: Creates the README.txt file.
  - Inputs: Café name

### What Changed
For Week 8, I expanded my Ever After Café Folder Maker by adding JSON functionality. I installed the Newtonsoft.Json NuGet package, created a CafeProject class to store project information, and added a SaveCafeData() procedure that creates a CafeData.json file. I also commented my functions and procedures, updated the README, and added the required Week 8 documentation.

### What the Tutorial Taught Me
The JSON tutorial showed me how applications can save information instead of losing it when the program closes. I learned how to use the Newtonsoft.Json package to serialize an object into JSON format and write that information to a file. I also learned how NuGet packages can add useful features to an application without writing everything from scratch.

### How JSON Applies to My App
JSON allows my Ever After Café Folder Maker to save important project information after the application finishes running. When the user enters a café name, the application creates a CafeData.json file that stores the café name, the project creation date, and the number of folders created. This makes the application more useful because the information can be saved for future reference instead of being lost when the program closes.
